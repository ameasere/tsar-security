| Version | Supported          |
| ------- | ------------------ |
| 1.0b    | :white_check_mark: |

## Reporting a Vulnerability

Reporting a vulnerability should include the following:

-Operating System, software/hardware specification, and more. This means Python version installed, library versions, Windows or other OS version.
-Brief description of the vulnerability, any libraries involved if any are. This could be an RCE, a buffer overflow, etc.
-Proof of concept
-Any suggested fix if you can recommend one.
